# StrataScratch_AnalyticalQuestions


This repository contains the SQL solutions for analytical questions solved on [StrataScratch](https://www.stratascratch.com/). 
The main goal of this repository is to showcase my SQL and analytical skills by solving real-world inspired data problems. 
Each question includes a detailed solution.

## Purpose

The purpose of this repository is to demonstrate my ability to:
- Analyze and manipulate data using SQL
- Solve practical analytical problems
- Apply SQL functions such as ROW_NUMBER(), MAX(), ABS(), CASE WHEN, etc.
- Use JOINs, conditional aggregation, and ranking for data analysis
- Document and explain problem-solving approaches clearly
